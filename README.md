## ⚙️ Установка и запуск

### 1. Создать базу данных в pgAdmin

1. Открыть pgAdmin → правой кнопкой на «Databases» → Create → Database
2. Имя: `vodokanal_db`
3. Открыть Query Tool (Tools → Query Tool)
4. Вставить содержимое файла `docs/schema.sql` и выполнить (F5)

### 2. Установить зависимости Python

```bash
cd vodokanal
pip install -r requirements.txt
```

### 3. Настроить подключение к БД

В файле `config.py` (или через переменную окружения `DATABASE_URL`):

```python
SQLALCHEMY_DATABASE_URI = 'postgresql://postgres:ВАШ_ПАРОЛЬ@localhost:5432/vodokanal_db'
```

### 4. Инициализировать и заполнить БД

```bash
# Создать таблицы через Flask
flask --app run init-db

# Заполнить тестовыми данными (30 товаров, поставщики, склады, документы)
flask --app run seed
```

### 5. Запустить приложение

```bash
python run.py
```

Открыть браузер: **http://localhost:5000**

## 📊 Функциональные возможности

### Справочники
- **Товары (номенклатура)** — полный CRUD, категории, единицы измерения, артикулы
- **Поставщики** — реквизиты (ИНН, КПП), контакты
- **Склады** — несколько складов, МОЛ

### Движение товаров
| Тип документа | Описание |
|--------------|----------|
| **Приход** | Поступление от поставщика на склад |
| **Расход** | Отпуск товара со склада |
| **Перемещение** | Передача между складами |
| **Списание** | Акт списания пришедших в негодность ТМЦ |

Каждый документ проходит цикл: **Черновик → Проведён → (Отменён)**  
Остатки изменяются только при проведении. Отмена откатывает остатки.

### Отчёты
- **Остатки на складах** — с фильтрами по складу и категории, общая стоимость
- **Оборотная ведомость** — движение за период
- **Дефицит** — товары ниже минимального запаса

### Дашборд
- Сводная статистика (товары, поставщики, дефицит)
- График движений за месяц (Chart.js)
- Последние документы
- Быстрые действия

## 🗄 Схема базы данных

```
users ──────────────────────────────────────────┐
categories ─┐                                   │
units ──────┤                                   │
            ▼                                   │
          products ──────── stock ─── warehouses│
                    \                           │
                     movement_items ── movements┘
                                    ── warehouses
                                    ── suppliers
```

### Таблицы (9 шт.)
| Таблица          | Описание                                |
|------------------|-----------------------------------------|
| `users`          | Пользователи системы, роли              |
| `categories`     | Иерархические категории номенклатуры    |
| `units`          | Единицы измерения                       |
| `suppliers`      | Поставщики с реквизитами                |
| `warehouses`     | Склады предприятия                      |
| `products`       | Товарные позиции (номенклатура)         |
| `stock`          | Текущие остатки (product × warehouse)   |
| `movements`      | Заголовки документов движения           |
| `movement_items` | Строки документов движения              |

### Представления (Views) для pgAdmin
- `v_stock_full` — полный отчёт по остаткам
- `v_turnover` — оборотная ведомость по месяцам
- `v_warehouse_summary` — сводка по складам
